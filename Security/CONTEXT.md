# Security Course Materials

## PDFs in this folder

### Main Course Materials

- **Leiden-26-Web-Privacy.pdf** | Web privacy, data protection, regulations
  European perspective on web privacy, privacy laws, and data protection principles for security professionals.

- **Les1IntroToSec3.pdf** | Introduction, fundamentals, security principles
  Foundational introduction to information security concepts and core security principles.

- **Les2RiskMgmtAndThreatModelingv2.0.pdf** | Risk management, threat modeling, vulnerability assessment
  Risk assessment methodologies and threat modeling techniques for identifying and prioritizing security vulnerabilities.

- **Les3SymmetricCryptography.pdf** | Symmetric encryption, algorithms, key management
  Symmetric cryptographic algorithms, encryption methods, and key distribution strategies.

- **Les4AsymmetricCryptography.pdf** | Asymmetric encryption, public-key cryptography, digital signatures
  Public-key cryptography, RSA, elliptic curve, and digital signature schemes for secure communication.

- **Les5NetworkSecurity.pdf** | Network protocols, firewalls, intrusion detection, VPN
  Network-level security including protocols, firewalls, IDS/IPS, and secure network architecture.

- **Les9WebSecurityI2.pdf** | Web vulnerabilities, OWASP, injection attacks, session management
  Common web security threats, OWASP Top 10, and vulnerabilities in web application development.

- **Les11AccessControlAndIdentityMgmt.pdf** | Access control, authentication, authorization, identity management
  Identity and access management systems, authentication mechanisms, and authorization frameworks.

- **Les12SecureComputationTechnologies.pdf** | Cryptographic protocols, secure computation, privacy-preserving tech
  Advanced techniques for secure multiparty computation and privacy-preserving computations.

- **Security2025_25-L6-application_security.pdf** | Application-level security, code security, vulnerability analysis
  Application security principles, secure coding practices, and vulnerability assessment methodologies.

- **Security2025_26-L7-application_security_defences.pdf** | Security defenses, mitigation strategies, secure development
  Security defense mechanisms and strategies for mitigating application-level vulnerabilities.

## Course Structure Overview

This collection covers:
1. **Foundations** - Security principles and risk management
2. **Cryptography** - Symmetric and asymmetric encryption
3. **Network Security** - Protocol-level defenses
4. **Web Security** - Application-level vulnerabilities
5. **Identity & Access** - Authentication and authorization
6. **Advanced Topics** - Privacy, defenses, and secure computation

Total: 11 core lectures + supplementary materials
