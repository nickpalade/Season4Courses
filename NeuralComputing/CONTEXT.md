## Course Overview

This course covers foundational neural computing topics from basic perceptrons through advanced architectures. The main components include: single-layer and multi-layer perceptrons with convergence proofs, Hopfield networks for associative memory using Hebbian learning rules, backpropagation for feed-forward networks, recurrent neural networks (RNNs), and convolutional neural networks (CNNs). The curriculum includes both theoretical foundations (stability conditions, energy functions, capacity limits) and practical exercises with detailed solutions.
