# MachineLearning Course Materials

## PDFs in this folder

- **lecture10-1.pdf** | Neural Networks, Backpropagation, AI History
  Introduction to neural networks covering biological inspiration, McCulloch-Pitts neuron, perceptron learning, and the development of backpropagation algorithms. Covers history from Cajal through Geoffrey Hinton's Nobel Prize work.

- **ML_Session_1_26-1.pdf** | ML Fundamentals, Supervised Learning, Feature Engineering
  Foundation course material on machine learning basics including supervised/unsupervised learning, predictive and descriptive tasks, model categories, and feature importance.

- **ML_Session_11_25_xAI.pdf** | Explainable AI, Model Interpretability, LIME
  Advanced session on making machine learning models interpretable and explainable, covering local vs global explainability methods, LIME techniques, and trust in predictions.

- **ML_Session_2_26-1.pdf** | Binary Classification, Performance Metrics, Ranking
  Covers binary classification tasks, confusion matrices, ROC curves, scoring and ranking methods, and class probability estimation.

- **ML_Session_3_26.pdf** | Multi-Class Classification, Regression, Clustering
  Beyond binary classification including multi-class approaches, regression methods, and unsupervised learning techniques like clustering and subgroup discovery.

- **ML_Session_4_26.pdf** | Decision Trees, Tree Models, Pruning
  Focuses on tree-based models including decision trees, tree growing algorithms, pruning strategies, and handling class imbalance issues.

- **ML_Session_5_26.pdf** | Linear Models, SVM, Kernel Methods
  Geometric classification approaches covering least-squares regression, perceptrons, support vector machines, soft margins, and kernel methods for non-linear separation.

- **ML_Session_7_25__2026_.pdf** | Feature Engineering, PCA, Data Preprocessing
  Comprehensive coverage of features and data handling: preprocessing, feature types (categorical, ordinal, quantitative), transformations, normalization, and principal component analysis.

- **ML_Session_7_26 - Copy.pdf** | Distance-Based Models, KNN, K-Means, Clustering
  Distance-based algorithms including nearest-neighbor classification, K-means clustering, medoid-based approaches, and hierarchical clustering evaluation methods.

- **ML_Session_8_26.pdf** | Probabilistic Models, Naive Bayes, Logistic Regression
  Probabilistic machine learning covering Bayesian vs frequentist perspectives, Naive Bayes classification, logistic regression, and Gaussian mixture models.

- **ML_Session_9_26.pdf** | Ensemble Methods, Boosting, Bagging, Random Forests
  Model combination techniques including ensemble methods, bagging, random forests, boosting algorithms, and bias-variance tradeoffs in experiments.
