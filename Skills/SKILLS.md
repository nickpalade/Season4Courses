# Study Skills Reference

## Available Commands

### `/study-quiz <course> <topic>`
5-question quiz with scoring and explanations.
```
/study-quiz ml decision trees
/study-quiz nc backpropagation
/study-quiz rl bandits
/study-quiz sec crypto
```

### `/study-explain <concept>`
3-tier explanation: intuition → formal definition → example.
```
/study-explain SVM
/study-explain Hopfield networks
/study-explain MDPs
/study-explain LIME
```

### `/study-cards <course> <topic>`
10 interactive flashcards with tracking.
```
/study-cards ml session 4
/study-cards nc hopfield
/study-cards rl mdp
/study-cards security web
```

### `/study-exam <course>`
10-question full mock exam across all topics.
```
/study-exam ml
/study-exam nc
/study-exam rl
/study-exam sec
```

## Course Shortcuts

- `ml` = Machine Learning
- `nc` = Neural Computing
- `rl` = Reinforcement Learning
- `sec` = Security
