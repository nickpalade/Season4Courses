## Course Structure Overview

This collection represents a comprehensive Reinforcement Learning curriculum from Leiden University. Materials progress from foundational concepts (MDPs, Bandits) through core methods (Dynamic Programming, Policy-based approaches) to advanced topics (Function Approximation, Model-based RL). The inclusion of psychology and neuroscience materials provides biological context for RL algorithms.
